# Quran Website API

مشروع بث مباشر لتلاوة القرآن الكريم تم تطويره بواسطة KIMO، يتيح لك إنشاء موقع احترافي لبث إذاعات القرآن بسهولة مع إمكانية التخصيص الكامل.

---

## المميزات

- بث مباشر لإذاعات القرآن الكريم
- واجهة بسيطة وسهلة التعديل
- إمكانية ربط API خارجي
- دعم التخصيص الكامل (UI / UX)
- قابل للنشر على أي استضافة
- كود مفتوح للتطوير والتعديل
